# Hunter

AI-powered autonomous job hunting and application assistant.

## Vision
Hunter helps automate:
- job discovery
- job matching
- ATS resume generation
- application workflows
- recruiter-friendly tracking

## Planned Features
- Job scraping
- AI JD analysis
- ATS score generation
- Resume customization
- Telegram notifications
- Approval-based auto apply
- Smart question handling
- Application tracking

## Tech Stack
- Python
- Supabase
- OpenRouter
- n8n
- Playwright
- Telegram Bot API
